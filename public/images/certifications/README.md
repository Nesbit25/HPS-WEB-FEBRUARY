# Certification Logos

Upload your certification logo images here with these filenames:

- `cert-logo-1.png` (or .jpg or .webp) - American Society of Plastic Surgeons
- `cert-logo-2.png` (or .jpg or .webp) - Second certification
- `cert-logo-3.png` (or .jpg or .webp) - Third certification

The system supports PNG, JPG, and WebP formats with automatic fallback.

## Recommended Image Specs:
- Format: PNG (transparent background preferred) or JPG
- Height: 200-300px (will be displayed at 64-80px tall, responsive)
- Width: Variable (maintain aspect ratio)
- File size: Under 200KB recommended
