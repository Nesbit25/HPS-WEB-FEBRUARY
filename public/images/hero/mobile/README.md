# Mobile Hero Images

## 📍 EXACT UPLOAD LOCATION
Upload your mobile hero images to this folder: `/public/images/hero/mobile/`

## 📏 IMAGE SPECIFICATIONS
- **Aspect Ratio:** 2:3 (Portrait)
- **Recommended Size:** 800x1200px or larger
- **Format:** JPG or PNG
- **File Size:** Under 300KB per image (compress if needed)

## 📝 FILE NAMING (EXACT NAMES REQUIRED)

Upload exactly 3 images with these exact names:

1. **`hero-slide-1.jpg`** - First hero slide (mobile home page)
2. **`hero-slide-2.jpg`** - Second hero slide (mobile home page)
3. **`hero-slide-3.jpg`** - Third hero slide (mobile home page)

## ⚠️ CRITICAL RULES
- File names are **case-sensitive**
- Do NOT change the file names
- Do NOT add spaces or special characters
- Use `.jpg` or `.png` extension (lowercase)
- Images will auto-rotate every 8 seconds on mobile devices
- These are **different** from desktop images (portrait orientation)

## 📤 HOW TO UPLOAD

### Via GitHub:
1. Go to: `https://github.com/Nesbit25/HPS-WEB-FEBRUARY/tree/hosted/public/images/hero/mobile`
2. Click "Add file" → "Upload files"
3. Drag your images (named exactly as above)
4. Commit changes
5. Wait 2-3 minutes for Vercel to rebuild

### Via Git Command Line:
```bash
git checkout hosted
git add public/images/hero/mobile/hero-slide-1.jpg
git add public/images/hero/mobile/hero-slide-2.jpg
git add public/images/hero/mobile/hero-slide-3.jpg
git commit -m "Update mobile hero images"
git push origin hosted
```

## 🎨 IMAGE CONTENT TIPS
- Use PORTRAIT orientation (tall, not wide)
- Crop for mobile viewing (face should be larger)
- Keep important content in center vertical area
- Avoid text or important details at top/bottom edges
- Test on actual mobile device if possible
