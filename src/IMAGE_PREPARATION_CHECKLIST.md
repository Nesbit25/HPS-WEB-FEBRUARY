## ✅ PRE-UPLOAD VERIFICATION

Before uploading to GitHub, verify ALL of these:

### File Naming:
- [ ] All file names are lowercase
- [ ] Dashes (-) not underscores (_)
- [ ] No spaces in file names
- [ ] File extensions are lowercase (.jpg not .JPG)
- [ ] Numbers are NOT zero-padded (1 not 01)

### Image Orientation:
- [ ] Desktop images are landscape (wide)
- [ ] Mobile images are portrait (tall)
- [ ] Logo works in both orientations

### File Sizes:
- [ ] Desktop hero: Under 500KB each
- [ ] Mobile hero: Under 300KB each
- [ ] SVG logo: Under 50KB
- [ ] PNG logo: Under 100KB
- [ ] Favicon: Under 10KB

### Image Quality:
- [ ] High resolution (sharp, not blurry)
- [ ] Professional lighting
- [ ] Appropriate compression (not over-compressed)
- [ ] Colors look correct
- [ ] No artifacts or noise

### Technical Requirements:
- [ ] Desktop hero: 1920×1080px or larger (16:9)
- [ ] Mobile hero: 800×1200px or larger (2:3)
- [ ] Logo SVG: Valid vector file
- [ ] Logo PNG: Transparent background
- [ ] Favicon: Square dimensions

### Content Guidelines:
- [ ] Desktop hero: Space on LEFT for text
- [ ] Mobile hero: Important content in CENTER
- [ ] Logo: Legible on navy background (#1a1f2e)
- [ ] No patient identifying information (HIPAA)
- [ ] Appropriate for public website

## 📤 UPLOAD CHECKLIST

When ready to upload:

1. **Prepare all files:**
   - [ ] 3 desktop hero images
   - [ ] 3 mobile hero images
   - [ ] 1 SVG logo (required)
   - [ ] 1 PNG logo (recommended)
   - [ ] 1 ICO favicon (optional)

2. **Navigate to correct GitHub folder:**
   - [ ] Desktop hero: `/public/images/hero/desktop/`
   - [ ] Mobile hero: `/public/images/hero/mobile/`
   - [ ] Logos: `/public/images/logos/`

3. **Upload files:**
   - [ ] Click "Add file" → "Upload files"
   - [ ] Drag and drop files
   - [ ] Write commit message (e.g., "Add hero images")
   - [ ] Click "Commit changes"

4. **Verify deployment:**
   - [ ] Wait 2-3 minutes for Vercel rebuild
   - [ ] Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
   - [ ] Check hero images rotate on home page
   - [ ] Verify logo appears in header
   - [ ] Test on mobile device

## 🚨 COMMON MISTAKES

Avoid these common errors:

❌ **Wrong file names:**
- `Hero-Slide-1.jpg` → Should be `hero-slide-1.jpg`
- `hero_slide_1.jpg` → Should be `hero-slide-1.jpg`
- `hero slide 1.jpg` → Should be `hero-slide-1.jpg`

❌ **Wrong orientation:**
- Desktop images in portrait → Should be landscape
- Mobile images in landscape → Should be portrait

❌ **Files too large:**
- 2MB desktop hero → Compress to under 500KB
- 800KB mobile hero → Compress to under 300KB

❌ **Wrong aspect ratio:**
- Square image for desktop → Should be 16:9
- Square image for mobile → Should be 2:3

❌ **Logo issues:**
- White background → Should be transparent
- Dark logo → Should be light/white colored
- Raster JPG logo → Should be vector SVG

❌ **Wrong folder:**
- Uploading to root → Should be in `/public/images/`
- Desktop in mobile folder → Check folder carefully
