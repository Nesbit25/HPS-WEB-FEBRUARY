# Desktop Hero Images

## 📍 EXACT UPLOAD LOCATION
Upload your desktop hero images to this folder: `/public/images/hero/desktop/`

## 📏 IMAGE SPECIFICATIONS
- **Aspect Ratio:** 16:9 (Landscape)
- **Recommended Size:** 1920x1080px or larger
- **Format:** JPG or PNG
- **File Size:** Under 500KB per image (compress if needed)

## 📝 FILE NAMING (EXACT NAMES REQUIRED)

Upload exactly 3 images with these exact names:

1. **`hero-slide-1.jpg`** - First hero slide (home page)
2. **`hero-slide-2.jpg`** - Second hero slide (home page)
3. **`hero-slide-3.jpg`** - Third hero slide (home page)

## ⚠️ CRITICAL RULES
- File names are **case-sensitive**
- Do NOT change the file names
- Do NOT add spaces or special characters
- Use `.jpg` or `.png` extension (lowercase)
- Images will auto-rotate every 8 seconds on the home page

## 📤 HOW TO UPLOAD

### Via GitHub:
1. Go to: `https://github.com/Nesbit25/HPS-WEB-FEBRUARY/tree/main/public/images/hero/desktop`
2. Click "Add file" → "Upload files"
3. Drag your images (named exactly as above)
4. Commit changes
5. Wait 2-3 minutes for Vercel to rebuild

### Via Git Command Line:
```bash
git add public/images/hero/desktop/hero-slide-1.jpg
git add public/images/hero/desktop/hero-slide-2.jpg
git add public/images/hero/desktop/hero-slide-3.jpg
git commit -m "Update desktop hero images"
git push
```

## 🎨 IMAGE CONTENT TIPS
- Use high-quality professional photos
- Ensure faces are well-lit and centered
- Leave space on the left side for text overlay
- Avoid busy backgrounds that compete with text
- Test images look good with dark gradient overlay (left side will be darker)
