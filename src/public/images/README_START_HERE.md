# 📸 Static Images System - Hero Images & Logos

This folder structure contains all the static images (hero banners and logos) for the Hanemann Plastic Surgery website.

## 📁 What Goes Where

```
/public/images/
├── hero/
│   ├── desktop/  ← Desktop hero images (1920×1080px, 16:9 landscape)
│   └── mobile/   ← Mobile hero images (800×1200px, 2:3 portrait)
└── logos/        ← Logo files (SVG required, PNG fallback, ICO optional)
```

## ✅ Files to Upload

### Required:
- [ ] 3 desktop hero images → `/public/images/hero/desktop/`
- [ ] 3 mobile hero images → `/public/images/hero/mobile/`
- [ ] 1 SVG logo → `/public/images/logos/logo-main.svg` ⚠️ **CRITICAL**

### Recommended:
- [ ] 1 PNG logo → `/public/images/logos/logo-main.png` (fallback)
- [ ] 1 ICO favicon → `/public/images/logos/favicon.ico` (browser icon)

## ⚠️ Important Notes

1. **File names are case-sensitive** - Must match exactly
2. **Desktop images are landscape** (16:9 wide)
3. **Mobile images are portrait** (2:3 tall)
4. **Logo must have transparent background**
5. **Deployment takes 2-3 minutes** after upload
6. **Clear browser cache** after deployment

---

## 🆘 Quick Help

**Images not showing?**
→ Check file names match exactly (case-sensitive)

**Wrong orientation?**
→ Desktop = landscape (wide), Mobile = portrait (tall)

**Logo not appearing?**
→ Ensure `logo-main.svg` exists in `/public/images/logos/`

**Still broken?**
→ See [HERO_IMAGES_LOGOS_UPLOAD_GUIDE.md](./HERO_IMAGES_LOGOS_UPLOAD_GUIDE.md) troubleshooting
