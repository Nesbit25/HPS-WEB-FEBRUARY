# Service Card Images

Upload your service category images here with these exact filenames:

## Required Images:

1. **Body.jpeg** (or Body.jpg, Body.png) - Body contouring procedures
2. **nose.png** (or nose.jpg, nose.jpeg) - Rhinoplasty procedures  
3. **face.png** (or face.jpg, face.jpeg) - Facial procedures
4. **breast.png** (or breast.jpg, breast.jpeg) - Breast procedures

## Image Specifications:
- Aspect Ratio: Portrait or Square (3:4 or 1:1 preferred)
- Dimensions: 800x1200px minimum
- Format: JPEG, PNG, or WebP
- File Size: Under 3MB recommended
- Content: Professional medical photography or stock images

## Important Notes:
- Filenames are case-sensitive (Body.jpeg must have capital B)
- If an image is missing, a gray placeholder will show
- Images will be used in the desktop carousel and mobile accordion
